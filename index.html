⚠️ 错误码参考

遇到错误？对照下表快速解决

错误码含义处理方式
401认证失败检查 API Key 是否正确、是否已被禁用或删除
403权限不足确认 Key 有权访问该模型；检查团队角色权限
429请求频率超限降低请求频率，或升级套餐提升 RPM 限制
402余额不足前往控制台充值后重试
400请求参数错误检查 JSON 格式、model 名称是否正确
404模型不存在确认 model 参数拼写；调用 /v1/models 查看可用列表
500服务内部错误稍后重试；若持续出现请提交工单
502上游网关错误上游服务暂时不可用，等待 30s 后重试
503服务过载当前并发过高，请稍后重试或联系客服扩容
504请求超时减少 max_tokens 或拆分长文本；检查网络
💡 所有错误响应均返回 JSON 格式：{"error":{"code":"xxx","message":"描述"}}，可据此做自动重试逻辑
